There are several ways of editing your application.

**Use Lovable**

Simply visit the [Lovable Project](https://lovable.dev/projects/0021db6e-9295-4afc-b6d4-32ce24123e91) and start prompting.

Changes made via Lovable will be committed automatically to this repo.

**Use your preferred IDE**

If you want to work locally using your own IDE, you can clone this repo and push changes. Pushed changes will also be reflected in Lovable.

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

**Edit a file directly in GitHub**

- Navigate to the desired file(s).
- Click the "Edit" button (pencil icon) at the top right of the file view.
- Make your changes and commit the changes.

**Use GitHub Codespaces**

- Navigate to the main page of your repository.
- Click on the "Code" button (green button) near the top right.
- Select the "Codespaces" tab.
- Click on "New codespace" to launch a new Codespace environment.
- Edit files directly within the Codespace and commit and push your changes once you're done.

## Key Features

### Advanced Multi-Page Portfolio System

The application includes a comprehensive portfolio building system with AI assistance, profile photos, SEO optimization, and interactive elements:

#### Template Preview System
- **Interactive Previews**: Click the preview button (eye icon) on any template card to see a live preview
- **Responsive Viewports**: Switch between desktop, tablet, and mobile views to see how templates adapt
- **Themed Example Data**: Each template includes example portfolio data that matches its unique style
- **Accessibility**: Full keyboard navigation, ARIA labels, and screen reader support

#### Profile Photo Management
- **Upload Profile Photos**: Add personalized profile photos to each portfolio
- **Fallback Support**: Automatically uses user profile photo if portfolio photo not set
- **Responsive Display**: Profile photos display beautifully across all templates with template-specific styling
- **Image Validation**: File type and size validation (max 5MB)

#### AI-Powered Content Generation
- **AI Assistant Panel**: Get intelligent suggestions for improving your portfolio
- **Content Generation**: Generate component content (header, about, skills, etc.) using AI
- **SEO Optimization**: AI-powered keyword generation and meta description optimization
- **Smart Suggestions**: Receive actionable recommendations for missing content and improvements

#### SEO Optimization
- **Meta Tags Management**: Set SEO title, description, and keywords
- **AI Keyword Generation**: Automatically generate relevant keywords based on portfolio content
- **SEO Analysis**: Get SEO score and recommendations for better search visibility
- **Character Counters**: Real-time feedback on optimal meta tag lengths

#### Multi-Page Navigation
- **Smooth Scroll Navigation**: Navigate between portfolio sections with smooth scrolling
- **Active Section Detection**: Navigation automatically highlights the current section
- **Mobile-Responsive Menu**: Hamburger menu for mobile devices
- **Sticky Navigation**: Navigation bar stays visible while scrolling

#### Interactive Elements
- **Smooth Animations**: Fade-in, slide-in animations for sections
- **Hover Effects**: Interactive hover effects on cards, buttons, and project items
- **Scroll Animations**: Elements animate into view as you scroll
- **Responsive Design**: All animations optimized for mobile devices
- **Accessibility**: Respects `prefers-reduced-motion` for accessibility

#### Empty State Handling
- **Helpful Guidance**: Clear messaging when portfolio is empty
- **Quick Actions**: Direct links to add components or use AI generation
- **User-Friendly**: Prevents confusion with empty portfolios
